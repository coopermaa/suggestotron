# Suggestotron

我的投票系統，照著 [RailsBridge - Intro to Rails](http://docs.railsbridge.org/intro-to-rails/) 這個課程一步步完成的。

[View the demo application here](https://suggestotron-c.herokuapp.com/)

## 基本功能

  * 使用者可以檢視主題
  * 可以對主題投票
  * 可以建立、編輯和刪除主題

## Bootstrap 美化

  * 用 bootstrap 套版讓網頁介面變漂亮
  * 使用 db/seeds.rb 自動建立種子資料
  * 為主題列表建立分頁
